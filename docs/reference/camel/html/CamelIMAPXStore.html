camel_imapx_store_ref_server ()

CamelIMAPXServer *
camel_imapx_store_ref_server (CamelIMAPXStore *store,
                              const gchar *folder_name,
                              gboolean for_expensive_job,
                              GCancellable *cancellable,
                              GError **error);

Returns the CamelIMAPXServer for store , if available.

As a convenience, if the store is not currently connected to an IMAP server, the function sets error to CAMEL_SERVER_ERROR_UNAVAILABLE and returns NULL. If an operation can possibly be executed while offline, pass NULL for error .

The returned CamelIMAPXServer is referenced for thread-safety and must be unreferenced with g_object_unref() when finished with it.
